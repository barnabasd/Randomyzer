## Have you ever spent hours searching for a random item giver mod that works with other mods? I sure have - that’s how this mod came to be, and so _Randomyzer_ lends an easy and customizable way to do just that!

### With features like:

*   **Mod compatibility** - if the items exist, _Randomyzer_ will be able to give it to you (even if said items aren’t even accessible with /give!)
*   **Multiplayer compatibility** - only the server host has to download the mod, and the others can use it too
*   An easy to use **configuration system** _(and soon even better!)_
*   Easy to understand and use **commands** 

***

### Base commands

*   **/randomyzer toggle:** start or stop the timer
*   **/randomyzer cycle:** instantly trigger an item drop
*   **/randomyzer config:** settings (details and examples in the wiki)

***

### Future plans:

*   Fabric and NeoForge support
*   Item filtering
*   Upgraded configuration system
*   Support for items with multiple states (for example: enchanted books, potions, suspicious stews, etc.)
*   Missing 1.21.x and 26.x versions
*   Fixing bugs **_(If you find a bug, please report it to us so we can fix it)_**

### (Possible) Questions&Answers  
  

**Q**: Will you backport the mod to X version?  
**A**: Not on our own, but if people ask for it, then possibly.  
  

**Q**: Will you update the mod to X version?  
**A**: Yes, we probably will.  
  

**Q**: Can I use this mod in my modpack/server?  
**A**: Since this isn't a survival friendly mod, the modpack will probably revolve around it, so as long as you give credit you are free to do so.
  

**Q**: Can I use this mod in my video?  
**A**: We would be happy if you did so, just please give credit for it. If you do make a video using the mod, feel free to message me on Discord (valaki\_0810), and we’ll probably link the video here.
